# Youtube Video Summarizer App

Youtube Video summarizer app summarises/highlights the key points in the given video.

## Tech Stack used:
1. LLM : Open Source LLM gemini-pro-1.5
2. Streamlit
3. Youtube Transcript API
4. Google Generative AI.
